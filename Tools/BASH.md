# Useful Stuffs

- `!!` : grab previous command, useful for `sudo !!`
- `!$` : grab previous command args
- `!:3` : grab third argument
	- `!:1-2`
- `echo $0` : shows shell running


- `grep -v -E '/proc | /sys'` : exclude /proc or /sys from output
